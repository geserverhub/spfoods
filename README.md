# SP Foods - Professional Company Website

เว็บไซต์บริษัท SP Foods สำหรับจำหน่ายอาหารสุขภาพและผลิตภัณฑ์เกษตรอินทรีย์

## ⚡ เทคโนโลยี

- **React 18** - UI Library
- **Tailwind CSS** - Trending CSS Framework
- **Vite** - Fast Build Tool
- **Lucide React** - Beautiful Icons

## 📋 ส่วนประกอบของเว็บไซต์

- ✨ **Header/Navigation** - เมนูหลักและการนำทาง
- 🎯 **Hero Section** - แบนเนอร์ประชาสัมพันธ์หลัก
- 🛍️ **Services** - แสดงบริการ 4 ประการ
- 📖 **About** - ข้อมูลเกี่ยวกับบริษัท
- 📞 **Contact** - ฟอร์มติดต่อและข้อมูลติดต่อ
- 🔗 **Footer** - ท้ายเพจพร้อมลิงก์โซเชียลมีเดีย

## 🚀 วิธีการเริ่มต้น

### 1. ติดตั้ง Dependencies
```bash
npm install
```

### 2. รันเซิร์ฟเวอร์พัฒนา
```bash
npm run dev
```

จากนั้นเปิด http://localhost:3000 ในเบราว์เซอร์

### 3. สร้าง Production Build
```bash
npm run build
```

### 4. ตรวจสอบ Build
```bash
npm run preview
```

## 📝 การปรับแต่ง

### เปลี่ยนชื่อบริษัท
แก้ไข `Header.jsx` และ `Footer.jsx` เพื่อเปลี่ยนชื่อจาก "SP Foods"

### เปลี่ยนสี
แก้ไขไฟล์ `tailwind.config.js` ในส่วน `colors`

### เพิ่มผลิตภัณฑ์
สร้าง component `Products.jsx` ใหม่และเพิ่มลงใน `App.jsx`

## 🚀 Development Tips

1. ใช้ Tailwind CSS classes สำหรับ styling
2. ใช้ Lucide React icons สำหรับไอคอน
3. ตรวจสอบการตอบสนองบนอุปกรณ์ต่างๆ
4. ทดสอบในเบราว์เซอร์ต่างๆ
